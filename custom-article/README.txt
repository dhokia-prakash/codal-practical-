Custom CRM

== Description ==
- Create custom post type
- Create custom meta box like phone number, email address and budget
- Create form with some parameter like update the name lable max lenght of input box.
- Form will be submit with ajax functionality.

== Short code ==
[custom-form name="Name" max_length="10" rows='5' cols='10']

- Using some parameter we can change the input lable name and input box length.
